PDKI Auto Fetch

Otomatisasi Pencarian Data PDKI

Cara Penggunaan

  1. Persiapan: Buka halaman PDKI Search di tab browser Anda
  2. Inisialisasi: Lakukan pencarian manual sekali (masukkan kode, klik cari, lalu klik detail hasil) - ini diperlukan untuk inisialisasi
  3. Input Kode: Masukkan kode pencarian di textarea di atas, satu kode per baris. Contoh: 
    1234567890
0987654321
1122334455
  4. Jalankan: Klik tombol Jalankan untuk memulai proses scraping. Tunggu sampai muncul notifikasi selesai
  5. Export Data: Setelah proses selesai, klik Export JSON atau Export CSV untuk mengunduh data hasil scraping

Penting:

  • Pastikan halaman PDKI Search sudah terbuka dan aktif di tab browser
  • Lakukan pencarian manual minimal sekali sebelum menggunakan ekstensi
  • Jangan tutup atau refresh halaman PDKI saat proses scraping berjalan
  • Jika ada kode yang gagal, akan ditampilkan di notifikasi akhir
